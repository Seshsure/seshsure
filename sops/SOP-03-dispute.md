# SOP 03 — DISPUTE HANDLING
**Trigger:** dispute task (ack due 24h; URGENT = client production stopped, jump the queue).
**Steps:** 1) Open the dispute — read both stories side-by-side. 2) If factory hasn't responded, nudge via their portal claim. 3) Check the client's photos against the lot's QC records. 4) Rule: root cause + scope required. Lot-wide only if evidence shows the defect spans the lot — this triggers proactive outreach to every client who received it. 5) Factory-fault replacement = $0 run auto-created; factory pays materials + freight.
**Done when:** status resolved/denied; invoice unpauses automatically.
**Never:** rule without a root cause (it corrupts scorecards); promise the client a resolution before ruling.
